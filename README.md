**[AEGIS, a Real-Time Global Intelligence Dashboard](https://github.com/AasthaShayla/aegis)**

A live "world monitor" that fuses free, open data onto one command-center map:
aircraft, satellites, earthquakes, natural events, weather, public cameras, global
news, markets, and cyber threats, with cross-stream correlation alerts and an
optional local-AI briefing. Runs key-free.
